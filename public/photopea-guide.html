📝 Step by Step Editing

1️⃣ Remove Unwanted Elements

Clone Stamp Method (S):

  1. Press S for Clone Stamp
  2. Set brush size: top slider
  3. Alt+Click on clean area
  4. Paint over unwanted area

Content-Aware Fill:

  1. Select area with Marquee (M)
  2. Edit → Fill
  3. Select "Content-Aware"
  4. Click OK

2️⃣ Color Correction

Adjust Brightness/Contrast:

  1. Image → Adjustments → Brightness/Contrast
  2. Move sliders
  3. Check "Use Legacy"
  4. Click OK

Highlight Gold:

  1. Image → Adjustments → Hue/Saturation
  2. Select color from dropdown
  3. Increase Saturation +20
  4. Adjust Lightness

3️⃣ Apply Filter

High-Pass Sharpen:

  1. Ctrl+J to duplicate layer
  2. Filter → Other → High Pass
  3. Set radius 1-3px
  4. Change blend mode to Overlay

Soft Glow:

  1. Ctrl+J duplicate
  2. Filter → Blur → Gaussian
  3. Radius 10-20px
  4. Change opacity to 30%

4️⃣ Fix Exposure & Sharpness

Fix Exposure:

  1. Image → Adjustments → Exposure
  2. Move Exposure slider
  3. Or use Levels (Ctrl+L)
  4. Adjust input sliders

Add Sharpness:

  1. Filter → Sharpen → Unsharp Mask
  2. Amount: 100-150%
  3. Radius: 0.5-1px
  4. Click OK

5️⃣ Portrait Enhancement

Background Blur (Bokeh):

  1. W key for Quick Select
  2. Select person/subject
  3. Ctrl+J to copy selection
  4. Select background → Gaussian Blur

Skin Retouching:

  1. J key for Spot Healing
  2. Click on blemishes
  3. For more: Frequency Separation
  4. (Advanced technique)

6️⃣ Scene Enhancement

HDR Effect:

  1. Filter → HDR
  2. Adjust Detail slider
  3. Adjust Highlights/Shadows
  4. Click OK

Orton Effect:

  1. Ctrl+J duplicate
  2. Filter → Blur → Gaussian (20px)
  3. Change blend to Soft Light
  4. Set opacity 70%
