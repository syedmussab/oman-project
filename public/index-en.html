🎭 Filter Effects

Apply creative filters like Vintage, HDR, or Artistic effects.

Filter 1: High-Pass Sharpening

1. Duplicate layer (Ctrl/Cmd + J)
2. Filter > Other > High Pass
3. Radius: 1-3 pixels
4. Set layer to "Overlay" or "Soft Light"

Filter 2: Soft Focus / Glow

1. Duplicate layer
2. Filter > Blur > Gaussian Blur (10-20px)
3. Change blend mode to "Screen" or "Lighten"
4. Lower opacity to 20-40%

Filter 3: Filter Gallery

Go to Filter > Filter Gallery. Explore folders:
• Artistic: Poster Edges, Water Color
• Sketch: Photocopy, Stamp
• Texture: Texturizer, Craquelure

Filter 4: HDR Effects

Filter > HDR > HDR Exposure. Adjust sliders for dramatic look with high detail & contrast.

☀️ Exposure, Sharpness & Noise Correction

Fix dark/bright photos, add sharpness, or reduce noise.

1. Exposure Correction

Go to Image > Adjustments > Exposure.
• Exposure: Adjust overall brightness (-3 to +3)
• Offset: Adjust mid-tones
• Gamma: Adjust contrast

2. Brightness/Contrast

Go to Image > Adjustments > Brightness/Contrast.
Increase Brightness for dark photos. Check "Use Legacy" for better results.

3. Sharpness

Go to Filter > Sharpen > Unsharp Mask.
• Amount: 100-150%
• Radius: 0.5-1 pixel
• Threshold: 0-3

4. Noise Reduction

Go to Filter > Noise > Reduce Noise.
• Strength: 5-10
• Preserve Details: 50%
• Reduce Color Noise: Checked

👤 Portrait Editing

Enhance portraits with skin retouching, subject isolation, and detail enhancement.

1. Select Subject

Press 'W' for Quick Select tool. Click and drag to select the person. Refine edges with "Select and Mask".

3. Background Blur (Bokeh)
2. Skin Retouching (Frequency Separation)

Advanced method:
1. Duplicate layer 2x (Ctrl/Cmd + J)
2. Bottom layer: Filter > Blur > Gaussian (2-5px)
3. Top layer: Image > Apply Image (Mode=Multiply)
4. Use Clone Stamp on top layer to smooth skin while preserving detail

3. Background Blur (Bokeh)

1. Select subject and copy (Ctrl/Cmd + J)
2. Select background, apply Gaussian Blur (5-20px)
3. Creates professional portrait look

4. High Pass Detail

1. Layer > Duplicate
2. Filter > Other > High Pass (1-2px radius)
3. Set blend mode to "Overlay"
4. Adjust opacity to enhance detail

🏛️ Scene & Building Enhancement

Enhance architectural photos with HDR, perspective correction, and lighting.

1. HDR Tone Mapping

Go to Filter > HDR > HDR Toning.
• AdjustDetail slider for structure
• Highlights for sky/clouds
• Shadows for architecture depth

2. Perspective Correction

Use Filter > Crop tool or:
1. Enable Ruler (Ctrl/Cmd + R)
2. Draw line along vertical edge
3. Image > Rotate Canvas > Arbitrary (+ correct degrees)

3. Orton Effect (Popular)

1. Duplicate layer
2. Bottom: Filter > Blur > Gaussian (20-30px)
3. Top: Filter > Blur > Gaussian (3-5px)
4. Top layer mode: "Soft Light"
Creates dreamy/luminous look

4. Architectural Straightening

For Nizwa Fort or Sultan Qaboos Mosque:
1. Crop tool > Auto (tick)
2. Or use Camera Raw filter
3. Adjust straightening slider
Keep vertical lines parallel!
