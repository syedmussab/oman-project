📖 Photopea Guide

🎨 Open Photopea

⌨️ Keyboard Shortcuts:

S Clone Stamp
B Brush
E Eraser
W Quick Select
U Hue/Saturation
Ctrl+U Color Dialog
Ctrl+L Levels
Ctrl+F Filter Gallery

1️⃣ Remove Element - Clone Stamp (S)

  1. Press S for Clone Stamp
  2. Set brush size from top slider
  3. Alt+Click on clean area to sample
  4. Paint over unwanted area

2️⃣ Color Correction

  1. Alt+U for Hue/Saturation
  2. Select color to adjust
  3. Move sliders
  4. For gold: increase saturation

3️⃣ Filter - High Pass

  1. Ctrl+J duplicate layer
  2. Filter > Other > High Pass
  3. Set radius 1-3px
  4. Change blend to Overlay

4️⃣ Sharpness

  1. Filter > Sharpen > Unsharp Mask
  2. Amount: 100-150%
  3. Radius: 0.5-1px
  4. Click OK

5️⃣ HDR Effect

  1. Filter > HDR
  2. Adjust Detail, Highlights
  3. Adjust Shadows
  4. Click OK

💾 Save for Website

  1. File > Export > WebP
  2. Quality: 80-90%
  3. Save to public/images/
  4. Use in website
