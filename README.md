# 🏛️ Oman Castles - Interactive Educational Website

An interactive educational website showcasing the rich heritage of Omani castles and cultural landmarks.

## 🌟 Features

- 📷 **Interactive Gallery** - 6 Omani castles with 3D hover effects
- 🎨 **Image Editor** - 12 filter effects (Bright, Dark, Vintage, B&W, etc.)
- 🎬 **Video Section** - YouTube integration with autoplay
- 🎮 **3D Interactive** - Interactive Khanjar (Omani dagger) with drag controls
- 🎵 **Audio Player** - Web Audio API synthesized music
- 🥚 **Hidden Easter Egg** - Press Ctrl+Shift+O for surprise!
- 📱 **Responsive Design** - Mobile hamburger menu
- 🌍 **Multi-language** - English, Arabic, Hindi support

## 🚀 Technologies Used

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (ES6+)
- Canvas API
- Web Audio API

## 🖥️ How to Run

1. Open `public/index-en.html` in any browser
2. Or use a local server:
   ```bash
   npx serve public
   ```
